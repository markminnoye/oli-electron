# Release Notes - v0.3.1 (oli-electron)

## Summary
This is a maintenance release that synchronizes the Electron app with the latest updates from the core webapp and updates project dependencies.

## Technical Improvements
- **Core Sync**: Updated the `app` submodule to include the latest UI refinements and stability fixes.
- **Dependency Updates**: Updated various npm dependencies in both the root project and the webapp submodule to their latest versions.
- **Build Process**: Re-built the distribution packages for macOS (arm64) to ensure compatibility with the latest codebase.
